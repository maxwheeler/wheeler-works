## What it does

- Wraps an Elgato capture device and a USB cassette deck behind a single CLI
- Runs an `ffmpeg` denoise + deinterlace pass with sensible defaults
- Generates a sidecar `.json` per recording with date, source, and notes
- Drops everything into a NAS-friendly directory layout

## Roadmap

Currently single-machine and macOS-only. I want to extract the capture step
into a small daemon so I can leave the rip rig running in the basement and
drive it from my laptop.
